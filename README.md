# eMerger
Simple script to clean update your system!

## Install
<ol>
    <li> Run <code>./setup.sh</code> </li>
    <li> Run <code>up</code> or go to <code>./src/</code> and run <code>emerger.sh</code></li>
</ol>

## Uninstall
<ol>
    <li>Run <code>./uninstall.sh</code></li>
</ol>
